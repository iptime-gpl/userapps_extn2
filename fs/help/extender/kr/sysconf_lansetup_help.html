• IP 주소
    • 익스텐더 자체의 IP주소입니다.
    • 변경시 익스텐더가 재시작 됩니다.
    • 공장초기화시 192.168.0.254로 설정됩니다.

  • 서브넷 마스크
    • 익스테더의 서브넷 마스크입니다.

  • 기본 게이트웨이
    • 익스텐더의 기본 게이트웨이입니다.
    • 이 값이 설정되어있어야 인터넷 통신을 통한 시간 설정등이 가능합니다.

  • DNS 주소
    • 익스텐더의 사용할 DNS주소값입니다.
    • 이 값이 설정되어있어야 인터넷 통신을 통한 시간 설정등이 가능합니다.
    • 한개만 입력하는 것이 가능합니다.

  • WPS연결시 IP주소 새로받기
    • WPS(EXT)버튼을 이용한 설정이 완료된 후, DHCP를 실행하여 IP주소구성을 받아옵니다.
    • 이 기능을 사용하면, IP주소/서브넷마스크/기본게이트웨이/DNS주소값이 변경될 수 있습니다.
    • 익스텐더가 할당받은 IP주소는 [ ipTIME 검색기 ]프로그램을 이용하여 검색할 수 있습니다.

  • 기본 무선 연결
    • 이 기능은 DUAL BAND를 지원하는 익스텐더 제품에서만 필요한 기능입니다.
    • 2.4GHz무선과 5GHz무선이 동시에 익스텐더 모드로 동작 시,
      • 유선포트는 "기본무선연결"과 연결됩니다.
      • 익스텐더의 설정페이지 접속은 "기본무선연결"로 가능합니다.
      • WPS버튼 사용하여 무선 연결 성공 시, "기본무선연결"은 해당 무선 연결로 설정됩니다.
