• 호스트 이름
    • 호스트 이름을 설정하면, IE등의 웹브라우저 상단의 타이틀 바에 이름이 표시됩니다.
    • ipTIME 검색기에서 검색시 표시됩니다.

  • 자동 설정 저장
    • 자동 설정 저장 기능을 '실행'시에는 모든 설정이 자동으로 저장됩니다.
    • 자동 설정 저장 기능을 '중단'시에는 설정 시에는 저장이 되지 않으며, 우측 상단의 [설정 저장] 버튼을 누를 경우 모든 설정이 저장됩니다.
  • 로그인 페이지 설정
    • 설정 화면 접속시 로그인 페이지를 보여줄지를 설정합니다.

  • 관리도구 접속방법
    • 로그인 페이지에서 관리도구를 클릭하였을때 접속 방법을 설정합니다.

  • 나이트LED모드
    • 제어가능한 LED를 제어할 수 있습니다.
    • 유선포트LED/EXT(Extending) LED/CPU(RUN) LED 등을 제어할 수 있습니다.
    • WPS연결 시에는 나이트LED모드는 동작하지 않습니다.

  • 공유기 재시작
    • 공유기가 재시작 됩니다.

  • 내부 네트워크 자동 변경
    • ipTIME 설치 도우미를 통한 설정 시 IP주소 변경을 허용합니다.
