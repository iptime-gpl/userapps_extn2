• 무선MAC 주소 인증
    • 무선 호스트의 MAC주소를 사용하여 특정 무선 호스트의 접속을 차단/허용할 수 있습니다.
    • 설정하기전에 무선 단말 기기의 MAC주소를 확인합니다.
    • 이 기능은 "무선 익스텐더 모드" 또는 "무선 AP모드"로 사용 시 사용할 수 있습니다.
    • "AP설정동기화"(무선고급설정 참조)기능을 이용하여 연결된 AP의 MAC주소인증설정을 가지고 올 수 있습니다.

  • 특정 무선 Client 접속 차단하기
    • 1. "MAC주소 인증" 체크박스를 체크합니다.
    • 2. "무선 연결 차단"을 선택합니다.
    • 3. 우측 하단의 "적용" 버튼을 클릭합니다.
    • 4. 차단시킬 MAC주소들을 "검색된 MAC주소"에서 찾아 체크합니다.
    • 5. "등록" 버튼을 클릭합니다.
    • 6. 등록할 MAC주소가 검색되지 않는다면, "MAC주소 수동으로 입력하기"를 사용하여 수동으로 등록합니다.

  • 특정 무선 Client 접속 허용하기
    • 1. "MAC주소 인증" 체크박스를 체크합니다.
    • 2. "무선 연결 허용"을 선택합니다.
    • 3. 우측 하단의 "적용" 버튼을 클릭합니다.
    • 4. 접속을 허용할 MAC주소들을 "검색된 MAC주소"에서 찾아 체크합니다.
    • 5. "등록" 버튼을 클릭합니다.
    • 6. 등록할 MAC주소가 검색되지 않는다면, "MAC주소 수동으로 입력하기"를 사용하여 수동으로 등록합니다.

  • MAC주소 수동으로 입력하기
    • 1. 등록할 MAC이 검색되지 않았거나, 무선MAC에 대한 설명을 추가하고 싶을때 사용합니다.
    • 2. 좌측 하단의 MAC주소 입력칸에 등록할 MAC주소를 입력합니다.
    • 3. MAC주소에 관한 설명을 입력합니다.
    • 4. "수동 등록" 버튼을 클릭합니다.
