• 설정한 시간에 무선 사용을 금지할 수 있습니다.
  • 금지된 시간에는 무선 신호가 자체가 출력되지 않습니다.
  • 무선 스케줄러는 익스텐더가 인터넷에 연결되어 타임서버로부터 현재시간을 얻어온 후 동작됩니다.
  • 인터넷에 연결하려면, [네트워크 설정]->[게이트웨이 및 DNS주소]가 올바르게 설정되어야 합니다.
  • "AP설정동기화"(무선고급설정 참조)기능을 이용하여 연결된 AP의 무선스케줄러설정을 가지고 올 수 있습니다.

  • 기본 설정 방법
    • 1. " 설정된 시간에 무선 사용 금지"를 선택한 후, 우측의 을 클릭합니다.
    • 2. " 설정 시간 추가하기"를 클릭합니다.
    • 3. 아래의 방식으로 시간을 입력합니다.
      • 시간입력 시 0-23 방식으로 입력합니다.
      • 자정은 0시로 입력합니다.
      • 시작시간 보다 끝나는 시간이 빠르면 끝나는 시간은 다음날로 인식됩니다.
        • 화 23:00 - 4:00 - 화요일 23시부터 수요일 오전 4시까지로 인식합니다.
    • 4. 설정할 시간을 입력 후, 버튼을 클릭합니다.
