• 동작 모드
    • 무선 익스텐더 모드(AP+Client) - 무선을 통한 무선 영역 확장 기능.
    • 무선 Client모드 - 무선을 통한 유선네트워크 확장.
    • 무선 AP모드 - 일반적인 AP기능.

  • 무선 익스텐더 모드
    • 1. 동작모드를 "무선익스텐더(AP+Client)"로 선택합니다.
    • 2. 우측 상단의 " AP검색"을 클릭하여 무선네트워크를 검색합니다.
    • 3. 검색이 완료된 후, 연결할 무선네트워크를 선택합니다.
    • 4. 연결할 AP이름/인증 및 암호화방식은 자동으로 설정됩니다.
    • 5. 내 무선네트워크 이름에 대한 설정 정책 를 선택합니다.
      • WPS(EXT)버튼을 통한 연결 시에도 위의 정책에 따라 "내 무선 네트워크 이름"이 설정됩니다.
    • 6. 연결될 AP의 네트워크 암호를 입력합니다.
    • 7. 우측 하단의 을 클릭합니다.
    • 8. 연결상태를 확인합니다.

  • 무선 Client 모드
    • 1. 동작모드를 " 무선 Client"로 선택합니다.
    • 2. 우측 상단의 " AP검색"을 클릭하여 무선네트워크를 검색합니다.
    • 3. 검색이 완료된 후, 연결할 무선네트워크를 선택합니다.
    • 4. 연결할 AP이름/인증 및 암호화방식은 자동으로 설정됩니다.
    • 5. 연결될 AP의 네트워크 암호를 입력합니다.
    • 6. 우측 하단의 을 클릭합니다.
    • 7. 연결상태를 확인합니다.

  • AP 모드
    • 1. 동작모드를 "무선AP"로 선택합니다.
    • 2. 내 무선 네트워크 이름을 입력합니다.
    • 3. 사용할 채널을 선택합니다.
    • 4. 사용할 네트워크 암호를 입력합니다.
    • 5. 우측 하단의 [적용] 버튼을 클릭합니다.

  • 참고1)WPS(EXT)버튼을 이용한 연결 설정시, 기존의 익스텐더 설정 내용은 새로운 설정으로 업데이트됩니다.
  • 참고2)WPS(EXT)버튼을 이용한 연결 설정시, 무선익스텐더모드로 설정됩니다.
